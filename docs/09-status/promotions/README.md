# Promotion records

Sem patří povinné záznamy přechodů mezi prostředími.

## Název souboru

`YYYY-MM-DD-<from>-to-<to>-<slug>.md`

Příklady:

- `2026-04-06-proto-to-test-user-management.md`
- `2026-04-07-test-to-prod-release-2026-w15.md`

## Povinnost

PR do `staging` nebo `main` bez nového záznamu v této složce neprojde policy guardem.

## Šablona

Použij `docs/10-templates/promotion-record-template.md`.
